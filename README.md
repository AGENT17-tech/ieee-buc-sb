## 📌 About This Repository

This repository serves as the **central archive** for all projects presented at the **IEEE BUC Student Branch Exhibitions**. Each exhibition is a semester-end showcase where student teams present functional engineering projects across multiple disciplines.

> **Purpose:** Document, preserve, and share the technical work produced by IEEE BUC members — serving as a portfolio for the branch and a reference for future members.

## 🚀 How to Submit Your Project

**Step 1 — Fork this repository**

**Step 2 — Copy the project template**
```bash
cp -r projects/template projects/[YEAR]-expo/your-project-name
```

**Step 3 — Fill in your project README** using the template

**Step 4 — Add your code, schematics, and media**

**Step 5 — Open a Pull Request** using the project submission template

> Full guide → [CONTRIBUTING.md](./CONTRIBUTING.md)
